# Ecommerce Website

A full-stack ecommerce web application built using modern web technologies.  
Users can browse products, add items to cart, place orders, and manage their shopping experience smoothly.

## Features

- User authentication
- Product listing and details
- Add to cart functionality
- Order management
- Responsive UI
- Backend API integration
- Database support

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript
- React.js

### Backend
- Java
- Java Servlet
- REST API

### Database
- MySQL
